## Design Context

### Users
独行的规则测试者，在桌面端快速验证一套无限流 CRPG 战斗规则。没有观众，也不需要讨好大众；界面的第一要务是准确传达数值与状态，同时在长时间调试中保持视觉耐读性。

### Brand Personality
**黄金 · 死亡 · 古朴**
- 黄金：消逝荣耀的残片，强调色如 tarnished gold（黯金）。
- 死亡：生存压迫感，界面应如石碑铭刻般庄重、不可违逆。
- 古朴：时间侵蚀的质感，拒绝光滑的科技感，拥抱纸页泛黄与石质粗粝。

### Aesthetic Direction
以《艾尔登法环》的 UI 气质为参考，但转换为**浅色主题**：不是黑暗地牢，而是“褪色者在黄金树遗迹中展开的泛黄羊皮卷”。
- **底色**：暖调 parchment（羊皮纸色）与 stone（石灰岩色），拒绝纯白。
- **强调**：古铜金（tarnished gold）用于交互与关键数值；干涸血（dried blood）用于敌人、危险与损失。
- **字体**：标题用庄严衬线 Cinzel，正文与数字用优雅衬线 Cormorant Garamond，营造铭刻感。
- **形态**：厚重边框、极小圆角、非对称布局。信息像刻在石板与古卷上，而不是浮在玻璃卡片中。

### Design Principles
1. **浅而不轻**：浅色背景不代表轻盈。用深色文字、厚重分割线与深沉投影压出分量。
2. **铭刻感排版**：大写/小型大写字母、宽字距、清晰的字号阶梯。让文字本身成为装饰。
3. **黄金与血**：古铜金引导希望与行动，干涸血标记威胁与伤害。避免任何蓝/紫/青色的“科技感”点缀。
4. **克制动画**：页面与元素以淡入/滑入登场（ease-out-quart），hover 有微弱光晕；禁止弹跳、弹性与无意义微交互。
5. **图标即符文**：game-icons 图标像古代符文般嵌入文字流中，不单独存在于圆角容器中。

### Anti-References
- 拒绝：深色模式 + 霓虹青/紫渐变（典型 AI 审美）。
- 拒绝：圆角大卡片网格 + 图标悬浮在彩色圆底上。
- 拒绝：玻璃拟态（glassmorphism）与发光边框。
- 拒绝：居中大标题 + 三列等宽功能卡片。
