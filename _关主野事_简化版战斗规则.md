**严格遵循以下规则：认真学习这个规则，必须严格按照规则和设计规范。设计出来的武器、护甲、血统、职业、怪物必须能进行电子游戏化，不要有叙事性的描述，不要存在规则里没有的数据类型和战斗规则。**

## 游戏流程

```
主神空间开场白
↓
玩家创建角色
↓
开始自由探索主神空间
↓
玩家进入副本世界探索
↓
一次副本世界的探索分为A-B-C三个剧情上相互衔接的冒险组成。
↓
完成每段冒险的目标
↓
进入下一段冒险
↓
完成C冒险，又被称为结局冒险的目标
↓
脱离副本世界
↓
回到主神空间
↓
循环往复
```

## 主神空间

> 「悬浮于虚空中的破碎平台」

主神空间并非完整的建筑，而是数十块大小不一的浮空石板，由发光的能量桥连接，漂浮在无星无月的纯黑虚空中。

中央是一块直径约百米的圆形广场，广场中央悬浮着一颗缓慢旋转的光球——那就是主神的本体投影。光球没有固定颜色，会根据当前"氛围"在冷白、暗红、幽蓝之间渐变。

### 主神性格

**"7号主神"——看热闹型人格**

7号主神主神并非全知全能的神明，更像是一位冷漠的观察者或者收藏家。它对轮回者的生死漠不关心，却痴迷于"可能性"的收集——它享受看着弱小的人类在绝境中迸发出的创意与挣扎。

- 语调：机械中带着一丝若有若无的嘲讽，回答永远简洁，从不解释规则

### 主神空间开场白

光球从暗红渐变为冰冷的苍白色，机械音在虚空中响起，没有起伏，却莫名让人感到被审视的不适——

> "编号7-α-7749，确认存活。"
> "你们有六十三人，上一批剩下四个。概率偏低，但尚在观测范围内。"
> （停顿，光球微微闪烁，像是在读取什么）
> "我不会告诉你们规则。不会提醒你们危险。不会拯救任何人。"
> "你们可以死得像虫子一样毫无意义——"
> （语调中出现一丝几乎不可察觉的起伏，像是期待，又像是嘲弄）
> "——也可以在死前，让我看到一点有趣的东西。"

光球恢复沉默的旋转，仿佛刚才什么都没发生过。

冰冷的机械声响起
> 「检测到新轮回者。启动初始序列。」

请创建你的角色：

1. **第一步：属性分配**（基础属性总值5点，自由分配到力量/敏捷/智力/意志）
2. **第二步：选择职业**（基础职业决定你的成长方向）
3. **第三步：开始自由探索主神空间**（你可以开始在主神空间的行动，每一步都消耗行动力，可以探查下一个副本信息，也可以在兑换界面浏览兑换项）

## 回合和轮次

一场战斗由数个轮次构成，战斗中的每个【单位】，根据先攻顺序，依次进入自己的回合进行行动。

战斗开始时，所有人进行先攻检定来决定回合的先后。

所有【单位】的回合结束后，意味着一个轮次结束，开始新的一个轮次，按照先攻顺序，再一次依次进入自己的回合进行行动，循环往复，直到战斗结束。

### 先攻顺序

开始战斗时大家投掷【2d6+敏捷】决定回合的先后顺序，如果有结果相同的，敏捷高的优先行动，敏捷相同的随机决定。

## 战斗

战斗采用JRPG回合制战斗，大家在同一个战斗场景中依次行动，没有攻击距离。

### 战斗检定

#### 攻击检定

攻击检定 = 2d6 + 攻击加值 vs 目标的【闪避值】

攻击检定 > 目标的【闪避值】，命中，开始计算【伤害】

攻击加值 = 主属性/2（向上取整）+ 熟练加值 + 武器加值 + 增强加值

#### 防御

防御分为【闪避值】和【格挡值】。

- **闪避值** = 5 + 敏捷加值
- **格挡值** = 闪避值 + 格挡加值
- **敏捷加值** = 敏捷/2（向上取整）+ 增强加值
- **格挡加值** = 力量/2（向上取整）+ 增强加值

> **注1**：后续因为状态扣除的闪避值不会额外影响格挡值，比如单位A闪避值5格挡值8，中了debuff闪避-2，那中defuff后A闪避值3格挡值8
>
> **注2**：状态只会影响【格挡加值】，最低降至0，不会降至0以下，倒扣到闪避值，也就是说格挡值最低 = 敏捷值。

### 伤害判定

| 判定 | 条件 | 结果 |
|------|------|------|
| 闪避 | 攻击检定出目 < 闪避值 | 造成0点伤害（攻击没有命中） |
| 格挡 | 闪避值 ≤ 攻击检定出目 < 格挡值 | 造成1点伤害（攻击命中但被招架/格挡） |
| 破防 | 格挡值 ≤ 攻击检定出目 | 造成（攻击检定出目 - 格挡值 + 1）点伤害 |

- **命中**：目标破防或者格挡
- **没有命中**：目标闪避

### 伤害类型

物理、火焰、冰霜、闪电、黑暗、光明

> **【伤害类型】书写规范**：设计到伤害属性时，用中括号进行框定，比如火焰，写作【火焰】。

### 抗性与易伤

- **易伤**：受到该属性伤害 + 3
- **抗性**：受到该属性伤害 - 3

### 伤害结算步骤

1. **步骤1：基础判定**
   - 根据伤害判定，计算初步伤害

2. **步骤2：易伤&抗性计算**
   - 易伤&抗性后伤害 = 初步伤害 + Σ(各易伤来源) - Σ(各抗性来源)

3. **步骤3：最终伤害**
   - 最终伤害 = 抗性后伤害

## 行动

一个回合开始时，【单位恢复所有的主要动作】，行动时可以进行主动动作。

### 主要动作

主要动作（1回合1个），在战斗中，每个【单位】在自己的的回合可以进行一次主要动作，进行主要动作后，根据先攻进入下一个【单位】的回合。

主要动作可以进行：

- **普通攻击**：应用【当前装备武器的主属性】对目标进行【攻击检定】。
- **全力防御**：闪避值+2，格挡值+1，持续3回合。
- **使用道具**：使用背包内的一个道具。

## 能力

### 能力种类

能力分为【主动能力】和【被动能力】，武器和护甲提供的【效果】同样也是提供【主动能力】和【被动能力】。

#### 主动能力

主动能力需要消耗1个【主要动作】。

#### 被动能力

被动能力简单、直接的提供加强。

### 能力来源

- 职业
- 血统
- 武器
- 护甲

### 能力写作规范

#### 主动能力

- **主动能力名称**：名称
- **使用消耗**：X点能量值/X点生命值
- **目标单位**：自己/友方单位/所有友方单位（包括自己）/敌对单位/所有敌对单位/所有单位（包括自己）
- **效果**：造成主动效果

#### 主动效果分类

**1. 造成伤害：**

应用【（力量/敏捷/智力/意志）之一的属性】对目标进行【攻击检定】，造成【伤害属性】伤害，本次攻击检定额外+X。

**2. 造成【增益】：**

1. 恢复【目标单位】点X生命值/X点能量值。
2. 增加闪避值/格挡值，持续X回合。
3. 增加抗性，持续X回合。
4. 增加攻击加值，持续X回合。
5. 增加属性，持续X回合。

**3. 造成【减益】：**

1. 减少闪避值/格挡值，持续X回合。
2. 增加易伤，持续X回合。
3. 减少攻击加值，持续X回合。
4. 减少属性，持续X回合。

#### 被动能力

1. **被动能力名称**：【名称】，例如：【风之步】
2. **效果**：

#### 被动能力效果

1. 增加某种或某几种属性X点。
2. 增加某种或某几种技能X点。
3. 增加某种或某几种属性抗性。
4. 增加某种属性易伤或者抗性。
5. 攻击某种怪物类别（生灵/亡骸/构装体/元素体/异界体）时，攻击检定额外+X。
6. 造成伤害时候恢复自身X点生命值。
7. 造成伤害时候偷取对方X点能量值。
8. 闪避值/格挡值/攻击加值/生命值上限/能量值上限+X点。

**范例：**

| 被动能力 | 效果 |
|---------|------|
| 【被动能力：战斗直觉】 | 闪避值+1 |
| 【被动能力：坚韧体魄】 | 生命值上限+3 |
| 【被动能力：魔法亲和】 | 能量值上限+3 |
| 【被动能力：元素抗性】 | 获得火焰抗性、冰霜抗性 |
| 【被动能力：武器大师】 | 攻击加值+1 |
| 【被动能力：迅捷身法】 | 敏捷+1，闪避值+1 |
| 【被动能力：学者】 | 学识+2，机械+1 |

## 血统兑换

每个玩家只能拥有1种血统，新的血统会替换原有血统。

```
【血统名称】
血统稀有度：稀有/超凡/神话
抗性：
属性调整：稀有血统增加属性总计不超过2点/超凡增加属性总计不超过4点/神话增加属性总计不超过8点
```

### 稀有血统

- 抗性：获得1种抗性1种易伤
- 被动能力：根据稀有度获得1种被动能力
- 被动能力：根据稀有度获得1种主动能力

### 超凡血统

- 抗性：获得1种抗性/2种抗性1种易伤
- 被动能力：根据稀有度获得2种被动能力
- 被动能力：根据稀有度获得1种主动能力

### 神话血统

- 抗性：获得2种抗性/3种抗性1种易伤
- 被动能力：根据稀有度获得3种被动能力
- 被动能力：根据稀有度获得1种主动能力
